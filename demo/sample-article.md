# 欢迎使用 WeChat Article Manager

这是一篇**示例文章**，展示如何使用 Markdown 编写微信公众号文章。

## 功能特性

WeChat Article Manager 支持以下功能：

- ✅ **Markdown 一键发布**: 从 Markdown 文件直接发布到微信公众号
- 🖼️ **自动图片处理**: 自动上传本地图片到微信素材库
- 📝 **草稿箱管理**: 创建、查看、删除草稿

## 文本格式

支持多种文本格式：

- **粗体文字**
- *斜体文字*
- ~~删除线~~
- `行内代码`

```javascript
// JavaScript 示例
```

```python
# Python 示例
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
```
